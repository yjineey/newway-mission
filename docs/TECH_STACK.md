# 기술 스택

## 핵심 기술
- **React**: UI 라이브러리
- **React Router**: 페이지 라우팅
- **Vite**: 빌드 도구 (빠른 개발 환경)
- **GitHub Pages**: 정적 사이트 배포

## UI/UX
- **스타일링**: CSS Modules / Tailwind CSS
- **아이콘**: Lucide React (통일된 벡터 아이콘)
- **다크모드**: CSS Variables 또는 styled-components theming
- **반응형**: Mobile-First Design (모바일 우선)

## UI 컴포넌트
- 댓글 시스템 (접기/펼치기)
- 더보기 메뉴 (Drawer/Modal)
- 하단 고정 네비게이션
- 다크/라이트 모드 토글

## 카테고리 그룹 분류
- **필수 정보**: 선교일정, 공지사항, 회의록, 비상연락망
- **교육/훈련**: 선교스쿨, 기도훈련, 말씀훈련
- **팀**: 팀빌딩, 팀미션, 찬양
- **준비/물품**: 준비물, 선교물품
