# 프로젝트 개요

## 뉴웨이교회 선교스쿨 웹앱

### 개요
2개 팀(이집트, 요르단)의 선교 준비 및 정보 공유를 위한 모바일 우선 웹 애플리케이션

### 주요 기능
- **팀별 콘텐츠 관리**: 이집트, 요르단 팀 분리
- **12개 카테고리**: 기본정보(4), 공동체(4), 준비사항(3), 기타(1)
- **실시간 업데이트**: Firebase 기반 동적 데이터
- **게시판 시스템**: 관리자 전용 글쓰기/수정/삭제
- **댓글 시스템**: 모든 페이지에서 댓글 작성 가능
- **첨부파일**: 문서, 이미지 업로드/다운로드
- **다크모드**: 라이트/다크 테마 지원
- **모바일 우선**: 반응형 디자인, 하단 네비게이션

### 기술 스택
- Frontend: React + Vite + Tailwind CSS
- Backend: Firebase (Firestore, Storage, Auth)
- Icons: Lucide React
- Deploy: GitHub Pages

### 팀
- **이집트 팀**: 이집트 선교 팀
- **요르단 팀**: 요르단 선교 팀

### 카테고리 구조
**기본정보 (4개)**
- 선교스쿨, 선교일정, 훈련일정, 비상연락망

**공동체 (4개)**
- 팀빌딩, 회의록, 찬양, 중보기도

**준비사항 (3개)**
- 준비물, 선교물품, 준비사항
