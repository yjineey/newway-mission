# 레이아웃 및 카테고리 구조

## 레이아웃 구조

### 메인 페이지 (`/`)

```
┌─────────────────────────────────┐
│  [로고]         Header      [≡] │  ← Header (우측 더보기 메뉴)
├─────────────────────────────────┤
│        [이집트] [요르단]          │  ← TeamSelector (2개 팀)
├─────────────────────────────────┤
│  📋 필수 정보                     │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐   │
│  │선교│ │공지│ │회의│ │비상│   │  ← CategoryGrid
│  │일정│ │사항│ │록 │ │연락│   │     (4열 배치, 라벨 그룹화)
│  │    │ │    │ │    │ │    │   │     아이콘: 통일된 벡터 아이콘
│  └────┘ └────┘ └────┘ └────┘   │
│                                 │
│  📚 교육/훈련                     │
│  ┌────┐ ┌────┐ ┌────┐          │
│  │선교│ │기도│ │말씀│          │
│  │스쿨│ │훈련│ │훈련│          │
│  │    │ │    │ │    │          │
│  └────┘ └────┘ └────┘          │
│                                 │
│  👥 팀                            │
│  ┌────┐ ┌────┐ ┌────┐          │
│  │팀빌│ │팀미│ │찬양│          │
│  │딩 │ │션 │ │   │          │
│  │    │ │    │ │    │          │
│  └────┘ └────┘ └────┘          │
│                                 │
│  📦 준비/물품                     │
│  ┌────┐ ┌────┐                  │
│  │준비│ │선교│                  │
│  │물 │ │물품│                  │
│  │    │ │    │                  │
│  └────┘ └────┘                  │
├─────────────────────────────────┤
│  [아이콘] [아이콘] [홈] [아이콘] │  ← Footer
│  선교일정 선교스쿨 홈  기도훈련   │     (5개 네비게이션)
│                 말씀훈련          │
└─────────────────────────────────┘
```

**더보기 메뉴 (우측 상단 ≡):**

```
┌──────────────────┐
│ 교회 소개         │  ← 클릭 시 모달/페이지
│                  │     (교회 정보 + 저작권)
├──────────────────┤
│ 🌙 다크모드 [⚪️] │  ← 토글 스위치
└──────────────────┘
```

### 카테고리 페이지 (예: `/mission-schedule`)

```
┌─────────────────────────────────┐
│  [로고]         Header      [≡] │  ← Header (우측 더보기 메뉴)
├─────────────────────────────────┤
│        [이집트] [요르단]          │  ← TeamSelector (현재 팀 표시)
├─────────────────────────────────┤
│                                 │
│  선택된 팀의 카테고리 콘텐츠        │  ← Content Components
│  (각 카테고리별 구조에 따라 표시)   │
│                                 │
│  ───────────────────────────    │
│  ⓒ 2025 뉴웨이교회              │  ← Content 영역 하단
│  ───────────────────────────    │
│                                 │
│  ┌──────────────────────────┐  │
│  │ [▼] 댓글 3개              │  │  ← 댓글 영역 (접기/펼치기)
│  └──────────────────────────┘  │     기본: 접혀있음
│                                 │
├─────────────────────────────────┤
│  [아이콘] [아이콘] [홈] [아이콘] │  ← Footer (현재 페이지 강조)
│  선교일정 선교스쿨 홈  기도훈련   │     (하단 고정 네비게이션)
│                 말씀훈련          │
└─────────────────────────────────┘
```

### UI 플로우

1. **메인 페이지**: 팀 선택 + 카테고리 그리드에서 선택
2. **카테고리 페이지 이동**: 선택된 팀의 해당 카테고리 콘텐츠 표시
3. **푸터 아이콘**: 어느 페이지에서든 다른 카테고리로 이동 가능

## 카테고리별 구조 상세

| 카테고리   | 세부설명             | 리스트 | 상세 페이지 | 글쓰기 | 형태            | 상세 내용                           |
| ---------- | -------------------- | ------ | ----------- | ------ | --------------- | ----------------------------------- |
| 선교일정   | 단기선교현황         | ❌     | ✅          | ❌     | 표              | 단기선교 일정, 떠나는 날, 항공 정보 |
| 공지사항   |                      | ✅     | ✅          | ✅     | 리스트 → 상세   | 리스트 클릭 시 상세, 글쓰기 버튼    |
| 선교스쿨   | 훈련일정표, 훈려지침 | ❌     | ✅          | ❌     | 일정표          | 교육 일정표 (상세 내역)             |
| 기도훈련   | 팀기도모임일정       | ✅     | ✅          | ✅     | 리스트 → 상세   | 리스트 클릭 시 상세, 글쓰기 버튼    |
| 말씀훈련   | 팀말씀훈련일정       | ✅     | ✅          | ✅     | 리스트 → 상세   | 리스트 클릭 시 상세, 글쓰기 버튼    |
| 팀미션     | 스터디 , 양화진방문  | ✅     | ✅          | ✅     | 리스트 → 상세   | 1주~6주 리스트 형식, 글쓰기 버튼    |
| 비상연락망 |                      | ❌     | ✅          | ❌     | 표              | 상세페이지 바로 (표 형태)           |
| 팀빌딩     |                      | ❌     | ✅          | ❌     | 표              | 상세페이지 바로 (표 형태)           |
| 준비물     |                      | ❌     | ✅          | ❌     | 체크리스트      | 상세페이지 바로 (체크리스트)        |
| 회의록     | 팀회의               | ✅     | ✅          | ✅     | 리스트 → 상세   | 리스트 클릭 시 상세, 글쓰기 버튼    |
| 선교물품   |                      | ❌     | ✅          | ❌     | 표 + 체크리스트 | 상세페이지 바로 (표, 체크리스트)    |
| 찬양       |                      | ❌     | ✅          | ❌     | 상세            | 상세페이지 바로                     |

**글쓰기 버튼 위치:** 리스트 페이지 우측 상단 (관리자만 표시)

**동적 데이터 관리:**

- Firebase Firestore에서 실시간으로 데이터 불러오기
- 글쓰기/수정/삭제 기능 지원 (리스트 페이지가 있는 카테고리)
- 정적 페이지는 관리자 페이지에서 수정
