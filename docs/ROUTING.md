| `/records` | 회의록 | ❌ | ✅ |
| `/missions` | 팀미션 | ❌ | ✅ |
| `/contact` | 비상연락망 | ❌ | ✅ |

| `/preparation` | 준비물 (리스트) | ✅ | - |
