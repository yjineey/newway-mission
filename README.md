# 뉴웨이교회 선교스쿨

> 2025 단기선교 훈련을 위한 웹 애플리케이션

## 프로젝트 소개

뉴웨이교회 단기선교팀의 선교 준비와 훈련을 지원하는 종합 관리 시스템입니다.
선교스쿨 일정, 팀별 훈련, 게시판, 연락망 등 선교 준비에 필요한 모든 정보를 한 곳에서 관리합니다.
